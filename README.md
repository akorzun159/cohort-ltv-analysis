# Cohort LTV Analysis / Анализ когортного LTV

[🇬🇧 English version below](#english-version)

---

## 🇷🇺 Русская версия

### 📊 Обзор проекта

Этот проект выполняет **когортный анализ пожизненной ценности клиента (LTV)** для интернет-магазина электроники. Анализ помогает маркетинговой команде понять, как меняется ценность клиентов в зависимости от:
- Месяца привлечения клиента (когорта)
- Категории товара первой покупки
- Платформы первой покупки

**Цель:** Рассчитать средний доход с клиента за первые 10 дней после первой покупки в разрезе различных сегментов.

---

### 🎯 Бизнес-контекст

Маркетинговой команде необходимо оценить:
- Какие маркетинговые кампании привлекают самых ценных клиентов
- Какие категории товаров генерируют наибольшую пожизненную ценность клиента
- Какие платформы приносят самых прибыльных пользователей

Этот анализ предоставляет данные для оптимизации маркетинговых бюджетов и стратегий привлечения клиентов.

---

### 📁 Описание данных

**Входная таблица: `orders`**

| Колонка | Тип | Описание |
|---------|-----|----------|
| `order_id` | int | Уникальный идентификатор заказа |
| `customer_id` | int | Уникальный идентификатор клиента |
| `order_date` | datetime | Дата создания заказа |
| `order_amount` | float | Сумма заказа в валюте |
| `platform` | string | Платформа заказа (mobile/desktop/app) |
| `category` | string | Категория товара |

---

### 📋 Техническое задание

| Требование | Реализация |
|------------|------------|
| Когорта определяется месяцем первой покупки | ✅ `cohort_month = first_order_date.dt.to_period('M')` |
| Клиент принадлежит только одной когорте | ✅ `groupby('customer_id').first()` |
| LTV рассчитывается за первые 10 дней (день 0-9) | ✅ `days_since_first_order < 10` |
| Окно анализа одинаково для всех клиентов | ✅ Одинаковый фильтр для всех |
| Срез по категории первой покупки | ✅ `first_category` из первого заказа |
| Срез по платформе первой покупки | ✅ `first_platform` из первого заказа |

**Выходные поля:**

| Поле | Описание |
|------|----------|
| `cohort_month` | Месяц первой покупки (формат YYYY-MM) |
| `first_category` | Категория из первого заказа клиента |
| `first_platform` | Платформа из первого заказа клиента |
| `cohort_size` | Количество клиентов в когорте |
| `avg_revenue` | Средний доход на клиента за первые 10 дней |
